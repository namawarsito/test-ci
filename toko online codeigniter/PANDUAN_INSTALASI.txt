

PANDUAN INSTALASI SOURCE CODE

1. Buka localhost/phpmyadmin pada web browser.
2. Buat database baru dengan nama 'project_tokoonline_ci'.
3. Import file 'project_tokoonline_ci.sql' ke database 'project_tokoonline_ci' yang sudah dibuat.
4. Copy folder 'project_tokoonline_ci' ke folder htdocs. 
	Letak folder htdocs XAMPP windows ada di C://XAMPP/htdocs
	Letak folder htdocs MAMPP osx ada di /application/MAMP/htdocs
5. Akses aplikasi dengan alamat 'localhost/project_tokoonline_ci' pada web browser.
6. Selesai.
